### Cara menjalankan code
Langkah langkah:
1. Clone git
   git clone https://github.com/Baru-san/ProjekFinal_VisKom.git
2. Membuka folder yang sudah di clone
   cd .../ProjekFinal_Viskom
3. Install libraries
   pip install cv2
   pip install cvzone
   pip install mediapipe
   pip install math
   pip install re
4. Run program
   python3 virtual_calc.py
   python3 beat_vir.py
